# Bibliography (Working)
**Project:** Life Subscription / Mundane Dystopia

This file is the canonical bibliography for the project. Use these entries (or their cite keys) rather than “sources consulted” placeholders.

## How To Cite In This Repo
- Prefer primary/official sources for quantitative claims (ONS, GOV.UK/DLUHC, Law Commission, FCA/FLA/Ofcom/BARB).
- When using a secondary source (media, think tanks, company reports), cite the underlying primary source as well (if it exists).
- For web sources, include an access date because pages change.
